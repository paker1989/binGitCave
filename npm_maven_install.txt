
/********************* npm install ****************************/

1. go:
https://docs.npmjs.com/getting-started/installing-node

2. download node js, then run:
 - node -v
 then run:
 - npm install npm@latest -g

 /*********************** Maven install ***************************/

1. go:
http://maven.apache.org/download.cgi

2. add JAVA_HOME variable

3. add {maven home}/bin in PATH

4. verify by  running:

- mvn -v
